# HO12 Sample 2 — Team KPIs Dashboard

## Your task
Your team's KPIs live in a shared sheet nobody opens. Build a hosted dashboard that shows the 4–6 key numbers leadership actually cares about.

## Metrics to show
- MRR
- Tickets resolved
- NPS score
- Churn rate
- New leads
- Pipeline value

## Chart type
KPI cards + line chart (trend over 3 months)

## How to build it
1. Open index.html in your browser
2. Replace the mock data arrays with your real data (or API call)
3. Update the KPI card values
4. Customise the chart labels and colours
5. Push to GitHub Pages for a live link
