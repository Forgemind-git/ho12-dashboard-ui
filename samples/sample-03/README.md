# HO12 Sample 3 — Content Performance Dashboard

## Your task
As a creator you cannot tell which content is actually working. Build a hosted dashboard from your content data.

## Metrics to show
- Views per post
- Engagement rate
- Follower growth
- Best-performing format
- Publishing cadence

## Chart type
Bar chart (views per post) + KPI cards

## How to build it
1. Open index.html in your browser
2. Replace the mock data arrays with your real data (or API call)
3. Update the KPI card values
4. Customise the chart labels and colours
5. Push to GitHub Pages for a live link
