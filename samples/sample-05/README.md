# HO12 Sample 5 — Project Status Dashboard

## Your task
Your projects' status is buried in a task tool and stakeholders keep asking where things stand. Build a hosted status dashboard.

## Metrics to show
- Projects by status (On Track / At Risk / Blocked)
- Tasks completed this week
- Upcoming deadlines
- Team workload

## Chart type
Donut chart (status split) + deadline list

## How to build it
1. Open index.html in your browser
2. Replace the mock data arrays with your real data (or API call)
3. Update the KPI card values
4. Customise the chart labels and colours
5. Push to GitHub Pages for a live link
