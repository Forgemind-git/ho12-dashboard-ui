# HO12 Sample 1 — Habit Tracker Dashboard

## Your task
You track habits, workouts and spending in scattered notes. Build a hosted dashboard that shows your weekly progress across 3–5 metrics on one page.

## Metrics to show
- Habit streak (days)
- Workouts this week
- Steps average
- Sleep average
- Spending vs budget

## Chart type
Bar chart (weekly) + KPI cards

## How to build it
1. Open index.html in your browser
2. Replace the mock data arrays with your real data (or API call)
3. Update the KPI card values
4. Customise the chart labels and colours
5. Push to GitHub Pages for a live link
